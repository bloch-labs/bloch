# User Guide

This track is for people using Bloch to build quantum algorithms. Start with installation and the 5-minute quickstart, then dive into tutorials and tooling.

```{toctree}
:maxdepth: 2
:caption: Users
```

## What you get here

- Installation matrix with prerequisites per platform.
- Quickstart to compile and run your first Bloch program.
- Tutorials that grow from single-qubit basics to full algorithms (Hadamard, Bell pairs, Deutsch–Jozsa, Grover, QAOA).
- Tooling: Bloch Studio, VS Code extension, CLI tips.
- FAQ and troubleshooting to unblock common issues.
