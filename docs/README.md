# Bloch documentation content

This directory holds the markdown source for Bloch docs. The Starlight site and deployment live in the separate `bloch-labs/docs` repo.

## Structure
- `index.md` – landing page with links to versions.
- `v1.1.x/` – current docs set (with class/import syntax in preview).
- `v1.0.x/` – previous stable docs set (functional style only).

## Usage
- Edit markdown here; keep any assets alongside the relevant version folder.
- When publishing, sync this folder into the site repo’s content path and build/deploy there.
