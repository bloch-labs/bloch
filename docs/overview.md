# Overview

Bloch is a modern, strongly typed, interpreted quantum programming language designed for hardware agnosticity and seamless integration of quantum and classical workflows. Bloch aims to *unify the quantum development stack* from source to OpenQASM.

This documentation covers the core language, runtime model, and tools.
