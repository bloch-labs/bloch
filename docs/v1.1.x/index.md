title: Bloch v1.1.x
---

Bloch 1.1.x extends the core language with a class system, modules, and the `@shots` annotation while keeping the original functional core. These docs cover the full 1.1.x surface as it exists on `main`.

- Start with [Getting Started](getting-started.md) to install and verify 1.1.x.
- Explore syntax and quantum flow in [Language Tour](language-tour.md).
- Learn the execution model in [Features](features.md) and the workflows in [Tooling](tooling.md).
- See the precise [Bloch API](bloch-api.md) for every type, keyword, annotation, gate, and class construct.
- Review how to [Contribute](contribute.md) responsibly to the 1.1.x line.
- Adapt real programs from `examples/` (see `examples/README.md`).

Classes, imports, `@shots`, and object lifetimes are first-class in 1.1.x. Use this line when you need modularity and richer program structure.
